# Moni-debts API

1. Clonar proyecto
2. ```npm install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levantar la BD
```docker compose up -d```

6. Levantar el Backend: ```npm start```

7. Ejecutar SEED (para llenar resetear y llenar la bd)
```
localhost:3000/api/seed
```
